
## Run Locally

Clone the project

```bash
  git clone https://github.com/iharshcheema/Student-result.git
```

Go to the Server directory

```bash
  cd Server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
Go to the Client directory

```bash
  cd Client
```

Install dependencies

```bash
  npm install
```

Start the Client

```bash
  npm run dev
```
